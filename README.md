CuCat
A clean, Apple-style landing page for CuCat Pet Care Food Products.

Overview
This repository contains a single-page site at index.html with:

Hero section
Responsive navigation
Product cards
Quality and feature sections
Customer stories
Call-to-action and footer
Usage
Open index.html in your browser.
Optionally run a local server:
python3 -m http.server 8080
Then visit http://localhost:8080.

Customize
Edit index.html to change headings, text, buttons, and colors.
Add images by placing them in the repo and using <img> tags.
